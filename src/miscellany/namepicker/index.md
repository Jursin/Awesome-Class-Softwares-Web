## 功能
- [x] 随机抽取
- [x] 可视化配置菜单
  <img src="./images/config.png" style="max-width: 50%; height: auto;">
- [x] 从外部读取名单
  - [在线名单编辑器](https://np-nameeditor.streamlit.app/)
  - [可视化名单编辑器](https://github.com/NamePickerOrg/NP-NameEditor)
  - 示例：（0=男，1=女）
    ```csv
    name,sex,no
    example,0,1
    ```
- [x] 特殊点名规则
- [x] 支持非二元性别
- [ ] 概率内定
- [ ] 悬浮窗（点击展开主界面）
- [ ] 软件内更新
- [ ] 同时抽选多个
