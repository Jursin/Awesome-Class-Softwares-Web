## 下载源码
- 克隆仓库到本地或直接[下载源码](https://github.com/ExamAware/ExamSchedule/releases)

## 部署
- 使用 Python 启动本地服务器

  ```bash
  python -m http.server 8000
  ```
  ::: tip
  或尝试使用 Python3
  :::

- 使用 GitHub Pages 部署

::: warning
注意，广播与看板的配置文件是**分开存储于两个文件夹之下的**
:::
