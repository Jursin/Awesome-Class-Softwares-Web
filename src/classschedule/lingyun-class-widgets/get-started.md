## 下载与使用

在 [Releases](https://github.com/Yamikani-Flipped/LingYun-Class-Widgets/releases) 页面下载最新版本压缩包，解压到合适的位置，运行 `LingYun_Class_Widgets.exe` 即可使用。

::: tip
可在系统托盘中找到凌云图标，进行设置、息屏等操作。
:::
