## 特性
- 对 Microsoft PowerPoint 和 WPS Office 有优化支持。
- **笔细的一头写字，反过来粗的一头是橡皮擦。（希沃白板并不支持）**
- 优化原版 Ink Canvas 手掌擦除墨迹的体验。
- 手绘的 **几何形状** 可以落笔后自动转换成 **印刷体** 。
- 完全重写 Ink Canvas 原版项目，添加众多新功能，添加 UI 交互动画，升级界面外观，同时也会修复大量问题，并优化使用逻辑与提升用户体验
