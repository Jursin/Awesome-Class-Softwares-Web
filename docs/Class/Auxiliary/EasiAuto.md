- **易用的设置界面**：前所未有的 ClassIsland 自动化编辑体验，简单几步即可配置完成
- **完备的登录方式**：具有 UIA 自动定位、OpenCV 图像识别定位、~~固定位置（暂未完成）~~ 三种登录方案，根据需求灵活切
- **运行前显示警告弹窗**：自动运行不再措不及防，还可暂时推迟登录
- **醒目的横幅警示**：兼具实用性与视觉冲击力，同时支持高度自定义
- **单次跳过**：暂时禁用自动登录，满足特殊场景下的灵活需求
- **错误重试**：无惧登录流程被打断
- **自动更新**：及时接收功能增强和问题修复
