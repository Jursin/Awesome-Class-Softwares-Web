## 基本编辑

- [x] 添加最多 11 条热点（后续更新可能会有所改变）
- [x] 长按词条序号可删除热点，或立即同步排行榜数据。
- [x] 直接编辑已有热点，允许设置是否保留词条热度。
- [x] 轻点词条右侧加热按钮可为热点加热，或长按以取消加热

## 个性化

- [x] 选取背景图片
- [x] 选取侧边栏中的头图
- [x] 编辑窗口标题，同时显示为侧边栏大标题

## 其它特色

- [x] 自动以非线性动画恢复窗口上次关闭的位置和大小
- [x] 流畅的交互，先进的 Material Design 3 设计风格
